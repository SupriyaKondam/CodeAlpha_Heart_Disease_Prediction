# Heart Disease Prediction using Machine Learning

## Project Overview
This project was developed as part of the CodeAlpha Machine Learning Internship (Task 4). The objective is to build a machine learning model that predicts the likelihood of heart disease using patient medical data.

## Objectives
- Analyze and preprocess the dataset.
- Train a machine learning classification model.
- Evaluate the model's performance.
- Predict the presence of heart disease.

## Dataset
Heart Disease Dataset containing patient medical attributes such as age, blood pressure, cholesterol, chest pain type, maximum heart rate, and other clinical features.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Workflow
1. Import Libraries
2. Load Dataset
3. Data Preprocessing
4. Exploratory Data Analysis
5. Model Training
6. Model Evaluation
7. Prediction
